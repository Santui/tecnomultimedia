Mi historia se basa en el universo de TLOZ (the legend of Zelda) donde Link cae en la isla de Quién(un lugar con ambiente 
de estilo de playa desertica) y a partir de ese lugar el usuario deberá elegir distintas opciones que se les va dando
para que Link pueda sobrevivir en ese lugar lleno de enemigos que ya conoce.
